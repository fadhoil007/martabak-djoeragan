# Logo Brand — Martabak Djoeragan Accu

Letakkan file logo PNG di folder ini. 
Untuk mengganti logo, cukup timpa file PNG-nya saja — kode tidak perlu diubah.

## File yang dibutuhkan (nama HARUS persis):
- tokopedia.png
- gofood.png
- shopee.png
- tiktok.png

## Rekomendasi:
- Format: PNG transparent background
- Ukuran: minimal 96x96px, rekomendasi 200x200px
- Rasio: bebas (akan auto-fit)

## Download logo resmi dari Wikipedia Commons:
- Grab:    https://commons.wikimedia.org/wiki/File:Grab_Logo.svg
- GoFood:  https://commons.wikimedia.org/wiki/File:Gofood_logo.svg
- Shopee:  https://commons.wikimedia.org/wiki/File:Shopee_logo.svg
- TikTok:  https://commons.wikimedia.org/wiki/File:Tiktok_logo_text.svg

Klik file → klik resolusi PNG yang diinginkan → Save As → rename sesuai nama di atas.
